# /tell <Badge type="info" text="LunaChat" />

`/tell <プレイヤー名> <メッセージ>`

- 指定したプレイヤーにメッセージを送信します．このメッセージは送信先のプレイヤー以外に知られることはありません．
- 送信先は同じサーバーにいるプレイヤーに限ります． (アルカディアからエデンのプレイヤーにメッセージを送信することはできません)
